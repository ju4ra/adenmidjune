Build for Test Deployment on [Arlink](https://asc_arlink.arweave.net/).

### Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
